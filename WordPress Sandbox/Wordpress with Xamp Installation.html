Step1 : Install Xamp

Step2 : Install Wordpress

Goto wordpress.org and download wordpress zip folder.
Unzip wordpress.zip file and you will get wordpress folder

Goto Htdocs file and Paste word press folder

Open Xamp installation location and goto inside htdocs folder. ie xamp/htdocs.
Paste wordpress folder

Step 3 Start Apache Server and Mysql from Xamp

Open xamp as "Run as Administrator".
Then start Apache Server and MySql server

Step 4 : Run xamp from url

        
            http://localhost:[port]/phpmyadmin
If you get phpmyadmin screen then its ok

Step 5:Setup Wordpress

Goto Url and type: 
        
            http://localhost:[port]/wordpress
This will run folder from htdocs by default.
Then click on next and fillout the form.

Wordpress Admin Url

        
            http://localhost:8080/wordpress/wp-login
            //Here I have used port 8080 .
            //Then username and password , my case : username:admin and password:admin

Wordpress Main Url
