Only Certain Field 
        
                COPY products(is_public, title, discount) TO 'D:\csv_backup\products_db.csv' DELIMITER ',' CSV HEADER;

postgresql Table to CSV

All Field of the table 
        
                COPY products TO 'D:\csv_backup\products_db.csv' DELIMITER ',' CSV HEADER;

Backup Postgresql File 

        
            1. Right click on database 
            2. Click on Backup w
            3. Give filename and select filetype:backup    
            4. Set Format : Tar 
            5. Click on Ok

Restore Postgresql File

        
            1. Right click on database 
            2. Click on Restore
            3. Find your backup file    
            4. Click on Restore
