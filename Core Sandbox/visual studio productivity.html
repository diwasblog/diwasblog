Important Visual Studio ShortCut

Shortcut Describption
Ctrl + Home Move to the first line of Code
Ctrl + End Move to the last line of Code
Ctrl + G Goto Particular Line 
Ctrl + ] Move the cursor to match Bracket
Ctrl + Right Arrow Move the cursor one word to the right
Ctrl + Left Arrow  Move the cursor one word to left
Ctrl + Enter Insert a blank line above the cursor
Ctrl + Shift +Enter  Insert a blank line below the cursor.
Ctrl + K , C Comments
Ctrl + K, U UnComments
Ctrl + Delete Delete the word right to the cursor
Ctrl + Backspace Delete the word left to the cursor
Alt + T Navigate in Tool menu

 

Shortcut Description
Ctrl + K, D Format the code
Ctrl + Shift + B Build the code
Ctrl + Shift + N Create New Project
Ctrl + M, O Collapse all methods
Ctrl + M, P Expends all methods
Ctrl + F Find 
Ctrl + H Replace
Ctrl + Shift + F Find  through out the project.
Ctrl + tab Navigate to open file
F9 Set a breakpoint
Ctrl + F9 Remove breakpoint on current line
Ctrl + Shift +F9 Remove all the breakpoint
F5 Run the code invoking Debugger
Ctrl +F5 Run the code without invoking Debugger
Ctrl+M, Ctrl + G Navigate View From Controller
